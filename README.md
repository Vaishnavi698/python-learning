# Python Learning Journey 🐍

A structured, topic-by-topic Python learning repository built from absolute beginner to job-ready fundamentals.

## What This Repository Covers

- **Foundations** — variables, dynamic typing, operators, conditionals, loops
- **Data Collections** — lists, tuples, sets, dictionaries, comprehensions
- **Functions** — parameters, return values, scope, lambda, map/filter
- **Object Oriented Programming** — classes, encapsulation, inheritance, polymorphism, abstract classes
- **Engineering Best Practices** — exception handling, file I/O (JSON/CSV), decorators, generators

## Structure

| File | Description |
|------|-------------|
| `learn.py` | Main learning file — all concepts with section comments |
| `test.txt` | Generated by file I/O exercise |
| `student.json` | Generated by JSON handling exercise |
| `students.csv` | Generated by CSV handling exercise |

## Approach

Each topic was:
1. Learned concept by concept with clear explanations
2. Practiced with hands-on exercises
3. Committed to git with meaningful messages — tracking real progress

## Tools Used

- Python 3.14
- VS Code
- Git + GitHub

## Goals

This repository is part of my preparation for a Python developer role — covering core language fundamentals, software engineering principles, and professional coding habits (version control, clean code, PEP 8 style).
